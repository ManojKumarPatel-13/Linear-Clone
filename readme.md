# Linear Clone

A frontend clone of Linear built with HTML and CSS for learning and practice.

## About

This project is a UI recreation of Linear, focusing on layout design, styling, responsiveness, and modern frontend development practices using only HTML and CSS.

The goal of this project is to improve my understanding of:

- Semantic HTML
- CSS Layouts (Flexbox and Grid)
- Responsive Design
- Component-Based UI Structure
- Modern Styling Techniques
- Clean Project Organization

## Features

- Responsive user interface
- Modern dashboard layout
- Sidebar navigation
- Issue management interface
- Clean and accessible markup

## Technologies Used

- HTML5
- CSS3

## Learning Objectives

- Practice building real-world interfaces
- Improve CSS architecture and organization
- Strengthen responsive design skills
- Replicate production-grade UI designs

## Disclaimer

This project is created for educational purposes only and is not affiliated with Linear.
